# 👗 Dressy 

## 📖 Sobre o Projeto
O Guarda-Roupa Digital é um aplicativo desenvolvido para ajudar os usuários a organizar suas roupas de forma prática e digital. O sistema permite cadastrar peças de roupa, armazenar informações em um banco de dados e visualizar todas as peças em um armário virtual.

Além disso, o aplicativo conta com a funcionalidade "Look do Dia", que sugere combinações simples utilizando as roupas cadastradas pelo usuário.

## 🎯 Objetivo
Facilitar a organização do guarda-roupa e auxiliar na escolha de combinações de roupas para o dia a dia.

## 🚀 Funcionalidades do MVP
- Cadastro de roupas;
- Edição e exclusão de peças;
- Visualização das roupas cadastradas;
- Organização por categoria, cor e estilo;
- Armazenamento dos dados no Supabase;
- Sugestão simples de "Look do Dia".
